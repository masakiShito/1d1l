## 日本語

- **アプリ名**: 1D1L

### 説明文（400〜800文字）
1D1Lは、忙しい日々でも続けられる「1日1行」日記アプリです。書く量を最小限にすることで、気持ちの整理や振り返りを自然な習慣にできます。シンプルな画面で迷わず記録でき、カレンダーや一覧から過去のメモを素早く確認可能。軽量・ミニマルな設計で、毎日の記録が続く体験にこだわりました。データは端末内に保存され、サーバー送信は行いません。アカウント登録も不要なので、すぐに使い始められます。今日の一言を積み重ね、1D1Lで日常を見つめ直しましょう。

### 主な機能
- 1日1行のシンプルな記録
- カレンダー表示で過去の振り返り
- 一覧表示でまとめて確認
- 端末内保存でオフライン利用

### キーワード

### プライバシー要約
- サーバー送信なし
- ログは端末内に保存
- アカウント不要

## English (Optional)

- **App Name**: 1D1L
- **Subtitle (~30 chars)**: A minimalist one-line diary
- **Tagline**: One line a day, effortlessly.

### Description (400–800 chars)
1D1L is a minimalist one-line diary designed for busy days. By keeping each entry short, you can reflect and stay consistent without pressure. The clean interface lets you write quickly, then review entries via calendar or list views. Everything is stored locally on your device—no servers, no accounts—so you can start instantly and keep your thoughts private. Build a daily habit with just one line and let 1D1L help you look back on your days.

### Key Features
- One-line daily journal
- Calendar and list views
- Offline, on-device storage
- Simple, distraction-free UI

### Keywords
diary, journal, one-line, minimalist, habit, reflection, notes, life log, calendar

### Privacy Summary
- No server transmission
- Data stored on device
- No account required
